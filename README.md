# HZDO11 🎮

Plataforma de jogos online gratuitos — sem instalação, direto no navegador.

## 🕹️ Jogos
- 37 jogos organizados em 8 categorias: Ação, Aventura, Puzzle, Corrida, Esporte, Clássico e Estratégia
- **Neon Dash** — runner cyberpunk exclusivo desenvolvido do zero

## ✨ Neon Dash
- 5 personagens desbloqueáveis com moedas (BOLT, KIRA, RYUU, SHADE, NOVA)
- 4 power-ups: Ímã, Escudo, 2x Moedas e Slow Motion
- Obstáculos, moedas e velocidade crescente
- Moedas salvas entre sessões

## 🛠️ Tecnologias
- HTML5, CSS3, JavaScript puro
- Canvas API para os jogos
- Zero dependências externas

## 📁 Arquivos
- `hzdo11.html` — site principal
- `neondash.html` — jogo exclusivo

## 🚀 Como usar
Baixe os dois arquivos na mesma pasta e abra o `hzdo11.html` no navegador.
Para colocar online, suba os dois arquivos juntos no Netlify ou GitHub Pages.
